== carto

Creates rpmap files for MapTool from TSV files generated by Donjon random dungeon generator

Before using, you must first 'gem install bundler' and 'bundle install'

Usage: ruby carto.rb -m map-file -o output_file -t tileset
    -m, --map-file [MAP_FILE]        TSV Map File
    -o, --output-file [OUTPUT_FILE]  Output rpmap file
    -t, --tileset [TILESET]          Tileset
